# 家庭保障体检报告

**一句话结论**
这个家已经有了初步的风险管理意识，但保障的根基还不够稳，目前最需要为两位成年人建立起能抵御大额医疗支出的安全网。

**这个家已经做对了什么**
1.  **为疾病风险打下了基础**：客户A本人已经配置了一份重疾险，说明家庭对重大疾病带来的收入损失风险有基础准备。
2.  **没有忽略任何一方**：配偶名下也已有一份保单，家庭在考虑保障时，没有只关注其中一人。

**谁最需要先被守住**
**两位成年人作为一个整体**。因为当前家庭最明显的共同缺口是缺少稳定的大额医疗报销保障。一旦发生住院治疗，家庭积蓄将直接面临冲击，这是需要优先为整个家庭筑牢的防线。

**家庭成员保障画像**
- **客户A (本人 / 成人女性)**
    - **已有基础**：拥有一份20万保额的重疾险，覆盖了核心的大病风险。
    - **待补强处**：缺少医疗费用报销和意外伤害保障。同时，作为家庭一员，其承担的家庭责任（身故/全残）也缺乏明确的兜底。
    - **一句建议**：在已有重疾基础上，优先补充医疗险，让保障更立体。

- **客户A丈夫 (配偶 / 成人男性)**
    - **已有基础**：根据产品名称，可能拥有一份出行或意外相关的基础保障。
    - **待补强处**：缺少明确的重疾保障和医疗报销保障，疾病风险是当前的保障空白。
    - **一句建议**：首要任务是建立重疾和医疗保障，这是家庭财务安全更核心的支柱。

**这个家最值得优先处理的 3 件事**
1.  **（最高优先级）为家庭建立医疗报销防线**：为两位成年人配置能覆盖住院费用的大额医疗险（如百万医疗险），这是转移大额医疗支出风险最直接的工具。
2.  **（次优先级）补齐配偶的重疾保障**：为丈夫配置一份重疾险，确保家庭中任何一员遭遇大病时，都有一笔定额补偿来维持生活。
3.  **（可观察优化项）审视家庭责任保障**：在补充上述基础保障后，结合家庭收入与负债情况，评估是否需要以及需要多少寿险（身故/全残保障）来兜底家庭责任。

**哪些地方可能花了冤枉钱**
当前未见明显的重复配置或资金低效情况。家庭保费主要集中于基础保障型产品，结构是健康的。

**保费压力提醒**
- **家庭年总保费**：15,000元。
- **保费压力分析**：由于缺少家庭年收入信息，无法计算保费占比，因此暂无法对保费压力程度做出判断。仅从总额看，处于常见家庭保费区间。

**建议先做的一步**
**共同行动**：可以一起了解并筛选适合的健康告知要求、能稳定续保的百万医疗险产品，为两位成年人同时配置上。这一步能最快地提升家庭整体抗风险能力。

**说明与边界**
1.  本报告基于提供的保单信息生成，保障状态与责任以正式合同条款为准。
2.  报告中提及的“新华保险畅行万里”保障责任，为基于产品名称的初步分类，请以保单条款确认为准。
3.  由于缺少年龄、具体收入、负债及子女/老人抚养责任等关键信息，本报告对保额充足度的判断仅为方向性参考。一份更精准的保障规划需要这些信息的补充。
4.  家庭保障总分为49分，属于“家庭风险偏高”区间，这主要反映了核心保障（医疗、部分成员重疾及身故责任）存在明显缺口，亟待补强。
