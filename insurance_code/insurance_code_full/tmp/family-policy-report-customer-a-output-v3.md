# 家庭保障体检报告

**一句话结论**
这个家已经有了两张保单，为未来做了准备，但保障之间还没有形成完整的闭环，最需要先把看病报销这件事安排好。

## 这个家已经做对了什么
1.  **为重大疾病做了准备**：客户A名下有一份20万保额的重疾险，这意味着如果发生合同约定的重大疾病，家庭能获得一笔定额的给付，用于应对治疗期间的收入损失或康复费用。
2.  **为意外风险做了安排**：丈夫名下的“畅行万里”保单，明确覆盖了从一般意外到多种交通、公共场景下的高额身故/全残责任，为家庭责任提供了一部分意外风险保障。
3.  **保费支出健康**：家庭年保费1.5万元，占年收入30万元的5%，这个比例在健康范围内，没有给家庭带来明显的财务压力。

## 谁最需要先被守住
**夫妻二人作为一个整体需要被优先保护。** 因为家庭处于双收入状态，且暂无子女、无房贷，当前家庭责任相对均衡。但两人都缺少住院医疗报销保障，这意味着任何一方生病住院，医疗费用都需要家庭自己承担，这是当前最需要共同补上的一层保护。

## 家庭成员保障画像
### 客户A（35岁，成人女性）
*   **已有基础**：已知有一份20万保额的重疾险，为重大疾病确诊后的定额给付打下了基础。
*   **还薄的地方**：**住院医疗报销保障是明确的空白**。因为重疾险解决的是确诊后一次性给钱的问题，不等于住院期间产生的医疗费用报销。所以，看病花的钱，目前还缺少分担的工具。
*   **一句建议**：在已有重疾给付的基础上，优先补充一份住院医疗险，解决医疗费报销的问题。

### 客户A丈夫（36岁，成人男性）
*   **已有基础**：有一份“畅行万里”两全保险，已知能覆盖疾病身故/全残以及多种场景下的高额意外身故/全残，同时带有一定的储蓄返还功能。
*   **还薄的地方**：**缺少重疾给付和住院医疗报销保障**。因为“畅行万里”的核心功能是身故/全残和特定意外保障，它不承担重大疾病确诊后的给付责任，也没有住院费用报销功能。
*   **一句建议**：现有保障偏重身故和意外，需要补充重疾险和医疗险，来应对大病治疗和住院花费的风险。

## 这个家最值得优先处理的 3 件事
1.  **（最高优先级）为夫妻二人补充住院医疗报销保障**：这是当前最确定的缺口。无论谁生病住院，都能通过保险报销大部分医疗费用，直接减轻家庭现金流压力。
2.  **（次优先级）为丈夫补充重疾给付保障**：他的现有保单不包含重疾责任。补充后，夫妻双方在面临重大疾病时，都能有一笔定额资金支持。
3.  **（可观察优化项）评估未来的家庭责任保障是否充足**：家庭目前无房贷、支出压力不大，但有赡养老人责任且准备要孩子。随着未来家庭责任加重，可以再回顾一下身故/全残类的保障额度是否足够覆盖未来的家庭开支。

## 哪些地方可能花了冤枉钱
当前未见明显的重复配置或资金效率问题。家庭现有的两张保单功能不同，分别覆盖了重疾和身故/意外，尚未形成功能重叠。

## 保费压力提醒
家庭年总保费为 **15,000 元**，占已知家庭年收入 **300,000 元** 的 **5%**。从比例上看，当前的保费支出处于健康水平，没有给家庭带来过重负担。

## 建议先做的一步
可以一起看看市面上常见的住院医疗险（例如百万医疗险），了解其报销范围和续保条件，先为夫妻两人把医疗费用报销这一层保障建立起来。
